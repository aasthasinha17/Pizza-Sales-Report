# **Project Overview**

This project highlights data analytics and business intelligence skills using SQL, Excel, Power BI, and Tableau through an in-depth analysis of pizza sales data. The goal was to derive meaningful business insights by examining customer purchasing behavior, sales performance, and operational trends.

## Dataset

Pizza Sales Dataset 2015 sourced from Kaggle.

## Objective

The project aimed to analyze key business metrics and create interactive visualizations to better understand sales performance, customer preferences, and overall business trends. The analysis helped identify high-performing products, peak sales periods, and areas requiring improvement.

## Analysis Performed

The following KPIs were analyzed to evaluate business performance:

* Calculated total revenue generated from pizza sales.
* Analyzed sales distribution across pizza categories and sizes.
* Computed average order value to understand customer spending patterns.
* Determined total pizzas sold and total number of orders placed.
* Evaluated average pizzas sold per order.

To gain deeper insights, multiple visualizations were created, including:

* Daily order trend analysis using bar charts to identify fluctuations in customer demand.
* Monthly, weekly, and hourly sales trends using line charts to identify peak business hours and seasonal patterns.
* Category-wise sales contribution and best/worst-selling pizzas using pie charts and comparative visualizations.

## Project Phases

### **Data Acquisition and SQL**

* Imported and managed the dataset in MS SQL Server.
* Used SQL queries to extract, clean, and analyze business metrics such as revenue, orders, and sales trends.

### **Excel Dashboard**

* Developed an interactive dashboard using pivot tables, slicers, and charts for exploratory analysis.

### **Power BI Dashboard**

* Designed a dynamic dashboard with interactive visuals and KPI tracking to present business insights effectively.

### **Tableau Visualization**

* Built advanced Tableau dashboards with customized charts, calculated fields, and trend analysis for enhanced storytelling.

## **Conclusion**

This project demonstrates the ability to transform raw data into actionable business insights through data analysis and visualization techniques. It reflects strong analytical, problem-solving, and dashboarding skills while showcasing proficiency in modern BI and analytics tools.
